HƯỚNG DẪN CÀI ĐẶT VBA CODE THỦ CÔNG
=====================================

Tôi đã tạo sẵn 3 file VBA code cho bạn:
1. Worksheet_Code.vba - Code cho sheet
2. ThisWorkbook_Code.vba - Code cho workbook
3. Module_Code.vba - Code cho module

BƯỚC 1: MỞ VBA EDITOR
=====================
1. Mở file Excel "QUẢN LÝ NHÀ NGHỈ.xlsm"
2. Nhấn Alt + F11 để mở VBA Editor
3. Hoặc vào tab Developer > Visual Basic

BƯỚC 2: CÀI ĐẶT CODE CHO WORKSHEET
==================================
1. Trong VBA Editor, tìm sheet "TỔNG HỢP THU - CHI NHÀ NGHỈ" ở bên trái
2. Double-click vào sheet đó
3. Mở file "Worksheet_Code.vba" bằng Notepad
4. Copy toàn bộ nội dung
5. Paste vào cửa sổ code của sheet
6. Nhấn Ctrl+S để lưu

BƯỚC 3: CÀI ĐẶT CODE CHO THISWORKBOOK
=====================================
1. Trong VBA Editor, double-click vào "ThisWorkbook"
2. Mở file "ThisWorkbook_Code.vba" bằng Notepad
3. Copy toàn bộ nội dung
4. Paste vào cửa sổ code của ThisWorkbook
5. Nhấn Ctrl+S để lưu

BƯỚC 4: TẠO MODULE MỚI
======================
1. Right-click vào tên project (VBAProject) ở bên trái
2. Chọn Insert > Module
3. Mở file "Module_Code.vba" bằng Notepad
4. Copy toàn bộ nội dung
5. Paste vào cửa sổ code của Module mới
6. Nhấn Ctrl+S để lưu

BƯỚC 5: ĐÓNG VBA EDITOR VÀ LÀM CHO MACRO HOẠT ĐỘNG
==================================================
1. Đóng VBA Editor (Alt + Q)
2. Lưu file Excel (Ctrl + S)
3. Đảm bảo file được lưu với định dạng .xlsm
4. Khi Excel hỏi về macro, chọn "Enable Macros"

KIỂM TRA HOẠT ĐỘNG:
==================

1. THỬ CHỨC NĂNG GHI THỜI GIAN:
   - Nhập một số vào cột C (ví dụ: C10)
   - Thời gian sẽ tự động xuất hiện ở cột D (D10)

2. THỬ CHỨC NĂNG TẠO BÁO CÁO:
   - Nhập một ngày vào cột A (ví dụ: A10)
   - Bảng báo cáo sẽ tự động xuất hiện ở cột H-I

3. THỬ CHỨC NĂNG DI CHUYỂN:
   - Đóng và mở lại file
   - Con trỏ sẽ tự động di chuyển đến dòng cuối + 20

CẤU TRÚC DỮ LIỆU CẦN THIẾT:
===========================
- Cột A: Ngày
- Cột C: Số tiền
- Cột D: Thời gian (tự động)
- Cột E: Loại (Thu/Chi)
- Cột F: Phương thức (Tiền mặt/Chuyển khoản)
- Cột H-I: Bảng báo cáo (tự động)

LƯU Ý QUAN TRỌNG:
================
- Code chỉ hoạt động trên sheet có tên "TỔNG HỢP THU - CHI NHÀ NGHỈ"
- Nếu tên sheet khác, hãy sửa dòng:
  allowedSheets = "TỔNG HỢP THU - CHI NHÀ NGHỈ"
  thành tên sheet thực tế của bạn
- Backup file trước khi cài đặt
- Đảm bảo macro được bật trong Excel

KHẮC PHỤC SỰ CỐ:
================
Nếu code không hoạt động:
1. Kiểm tra macro đã được bật chưa
2. Kiểm tra tên sheet có đúng không
3. Kiểm tra code có được đặt đúng vị trí không
4. Nhấn Ctrl+Break để dừng macro nếu gặp lỗi
